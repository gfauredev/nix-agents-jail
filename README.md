A Nix flake based sandbox to run LLM agents securely, using [jail.nix] and
[llm-agents.nix]. To be imported in project specific flakes and given the
necessary packages. Based on [Anderson J’s] flake.

[Anderson J’s]: https://dev.to/andersonjoseph/how-i-run-llm-agents-in-a-secure-nix-sandbox-1899
[jail.nix]: https://git.sr.ht/~alexdavid/jail.nix
[llm-agents.nix]: https://github.com/numtide/llm-agents.nix
